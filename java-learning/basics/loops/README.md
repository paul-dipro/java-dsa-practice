# 🔁 Loops

This folder contains Java programs focused on practicing **loop constructs** — one of the most fundamental building blocks in programming.

## 📁 Files

| File | Description |
|------|-------------|
| `JavaLoopII.java` | Demonstrates various loop types in Java — `for`, `while`, and `do-while` — through practical examples |
| `MultiplicationTable.java` | Generates a formatted multiplication table using nested loops |

## 🧠 Concepts Covered

- `for` loop
- `while` loop
- `do-while` loop
- Nested loops
- Loop control with `break` and `continue`
- Pattern-based output using loops

## 🚀 How to Run

Make sure you have the [JDK](https://www.oracle.com/java/technologies/downloads/) installed, then:

```bash
# Compile
javac JavaLoopII.java
javac MultiplicationTable.java

# Run
java JavaLoopII
java MultiplicationTable
```
